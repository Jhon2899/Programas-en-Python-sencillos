 1. Diseñe un programa que reciba un número entero positivo del usuario y diga si es un número par o impar.  
 ```Pyhton
 num = int (input("Ingrese un numero: "))

if num % 2 == 0:
 print("El numero es par")
else: 
 print("El numero es impar")
 ```
